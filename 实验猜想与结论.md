# 实验猜想与结论

> 本文档沉淀 `迭代模型/对比` 项目实验至今的猜想与结论，**严格区分"已验证结论""待验证猜想""方法论反思"**。五测后再次修订：澄清 HIO 收敛的概率性（周期斜线=非收敛态）、推翻 C7（sigmoid 慢相变非自锁）、坐实 relaxed HIO 修复尺度爆炸、修正 R5（ssim 在 HIO 场景失效）。实验过程见 [初测汇报.md](初测汇报.md)、[二测汇报.md](二测汇报.md)、[四测汇报.md](四测汇报.md)、[五测汇报.md](五测汇报.md)。

> ⚠️ **全局前提（C6）**：UNet 实验受 CUDA 非确定性影响，**单次结果有随机性**。五测 sigmoid 长跑给出最强证据：同代码四测 3000 轮 ssim 0.101、五测 2000 轮 ssim 0.95。本文结论方向性可靠但绝对数值待多次跑取统计确认。

## 1. 已验证结论（有硬实验支撑）

### C1. 甲方 HIO 实现正确
真值 support 下漂亮收敛：ssim 0.95 / amp_cc 0.93 / IoU 0.77（初测）。严格 HIO 公式、FFT/IFFT、振幅替换、HM 实现无误。

### C2. HIO 收敛是概率性事件（五测修正：非"绝对不收敛"）
四测给 HIO 10000 轮 ssim 0.014，判"不收敛"。**五测修正**：HIO 不是绝对不收敛，而是**概率性收敛**——收敛则出形态轮廓、不收敛则出周期斜线，与迭代长度无单调关系（短跑可能收敛、长跑可能漂走）。收敛与否由随机初始相位+迭代轨迹决定。四测/五测长跑恰好都没收敛（斜线）。对策：多次短跑撞收敛（见 C10、R10）。support 估计的"鸡生蛋"循环依赖仍是阻碍收敛的因素之一。

### C3. UNet 相位恢复（四测修正：配准后非"全未恢复"）
配准后实验3（tanh+HIO）在 it=340 达 **ssim 0.944**（形态恢复峰值，配准后真实值，与人眼看图一致，非暗背景虚高）。**配准前 amp_cc≈0 / Δφ=π/2 的"四版本全未恢复"判断是平移歧义假象**——配准后露出部分恢复。但实验3 末轮尺度发散（见 C8），未稳住。

### C4. sigmoid + HIO 反馈塌缩到平凡解
sigmoid（恒正）+ HIO 反馈，2000 轮：amp_cc=0.000、IoU=0.000，psnr/ssim 飙高是暗背景虚高假象，网络输出塌成近零/平坦图。

### C5（新）. 配准是评估前提（消除平凡歧义）
相位恢复有固有的**平凡歧义**——整体平移、共轭反转（twin image）、全局相位因子，三者傅里叶振幅完全等价。未配准时：Δφ 恒为 π/2、amp_cc→0 是**物体平移的特征签名**，与"完全没恢复"无法区分。四测配准后 amp_cc 从 ~0.09 升到 ~0.25，露出真实恢复。**不配准的指标无意义**。配准方法（FFT 互相关 + 歧义群枚举）经 sanity check 验证可靠（构造已知变换，配准内积 1.000）。来源见 [调研报告.md](调研报告.md) §5.1/§5.2 + [28-30]。

### C6（新）. CUDA 非确定性，实验不可复现
同代码、同 seed（UNet seed=0、相位 seed=42）、同机器，UNet 实验跑两遍 loss 从 it=20 就分叉（2.2138e-5 vs 2.1819e-5）。根因是 cuDNN 卷积/反传的非确定算法 + DIP 对初始轨迹的敏感。**迭代流程未变（git diff 客观确认），差异完全来自 GPU 非确定性**。这动摇了初测/二测所有单次 UNet 结论的稳健性。

### C7. ~~support 自锁（实验2 sigmoid+置0 的病根）~~ —— 五测推翻
四测判 sigmoid+置0「support 自锁死锁」（3000 轮 ssim 0.101）。**五测推翻**：sigmoid 长跑 2000 轮 ssim 0.95、看图效果好，780–980 轮有明显相变。**不是死锁，是慢收敛相变**——能否触发高度依赖随机性（强证 C6）。

### C8（新）. tanh+HIO 尺度发散（实验3 末轮）
实验3（tanh+HIO）末轮 psnr 走到 −38（rho 数值爆炸）。机制：① HIO 反馈是累加更新 `current_input − β·raw`，raw 有持续偏置则单调发散；② support 外无约束（HM 只在 support 内，置0 才有界，实验2 置0 不发散正是因此）；③ tanh 只限制网络输出 raw∈[−1,1]，不限制迭代状态 rho_next；④ 振幅 loss 只约束 FFT(raw×support) 振幅，不约束 rho_next 尺度。**loss 一直降、amp_cc 归一化平稳，但 rho_next 早已发散**。

**解法（relaxed HIO，五测已验证有效）**：support 外改 `γ·ρ−β·ρ′`（γ<1），把累加变成几何级数、收敛到有限稳态 `−βr/(1−γ)`，防发散（见 [调研报告.md](调研报告.md) §2.3、[31][32]）。受 RAAR（Luke 2005）松弛思路启发的工程简化，非严格 RAAR。实验1/3 同用，γ=0.9。**五测长跑验证：实验3 末轮 psnr 不再走负（−38 → 4.85），rho 不再爆炸。** 4.85 偏低是末轮尺度漂移低点（ssim 仍 0.93，形态在），非发散。

### C9（新）. amp_cc 在本 setup 不可靠
amp_cc 量的是"振幅谱拟合度"，和形态恢复脱钩：实验2 amp_cc 0.30（loss 直接优化振幅，形态却锁死）；纯随机图配准后 amp_cc 也能到 0.2（基线）；实验3 amp_cc 0.15（低于基线，尺度发散扰动频域）。**amp_cc 在 0.15–0.30 区间横向比较三个实验基本无区分力**。

### C10（新）. HIO 收敛概率性 → 周期斜线是非收敛态
HIO 单次跑结果二选一：收敛→形态轮廓，不收敛→周期斜线（开HM）或平坦（关HM）。与迭代长度无单调关系（HM 消融 A 开HM 400轮收敛出轮廓、五测长跑 HIO 13000轮没收敛出斜线）。收敛由随机初始相位+轨迹决定。对策：多次短跑撞收敛（R10）。详见 [五测汇报.md](五测汇报.md) §4.4。

### C11（新，待查）. UNet+tanh+HIO 的 twin image
长跑 UNet+tanh+HIO 结果：中间一个好图 + 上下各半个偏移轮廓卡在画布边界。疑似 support 外 HIO 反馈或 FFT 卷绕引入的共轭像。五测仅记录，未查。

## 2. 核心猜想（四测后状态）

### H1. 纯振幅 loss 锁不住相位（修正）
- **原表述**（过于绝对）：相位完全锁不住，Δφ 卡 π/2 = 相位随机。
- **四测修正**：纯振幅 loss **能恢复低频形态**（实验3 it=340 配准后 ssim 0.944），但**锁不住高频相位**（Δφ 全程 π/2，因弱振幅频率无优化动力）。且 support 策略（自锁 vs 演化）是关键变量，不止 loss 本身。
- **状态**：部分成立，表述需细化。

### H2. HIO 反馈要求 ρ′ 振荡（实验3 支撑）
HIO 反馈 `ρ_k−β·ρ′` 要靠 support 外正负振荡抵消才不发散；tanh 输出虽可负但网络无动力振荡 → 累加发散。实验3 尺度发散（C8）支撑此猜想。

### H3. 网络输出不天然振荡
网络先验偏向平滑自然图，support 外不振荡；即使 tanh 允许负，也无动力画振荡。实验3 支撑。

### H4. 软学习 + HIO 反馈不兼容（部分推翻）
- **原表述**：HIO 反馈在软学习里有害。
- **四测修正**：HIO 反馈**让 support 能演化**（实验3 it=340 形态恢复，优于实验2 自锁），并非完全有害；但**引入尺度失控**（C8）。应改为"HIO 反馈打破 support 自锁、让形态能长，但代价是尺度发散"。
- **状态**：部分推翻，需重新表述。

## 3. 方法论反思

### R1. 前期 UNet 版本对比不严格
不同版本跑不同轮次、同时改多个变量。四测在配准 + 同 wall-clock（HIO 10000 / UNet 3000，各 ~17 min）下重新对比，才公平。

### R2. 区分"猜想"与"结论"
前期把"Δφ=π/2 即锁不住""HIO 反馈有害"当结论，实为猜想，被四测（配准后）部分推翻。今后严格区分。

### R3. "是否用 HIO 反馈"作为实验变量
不预设好坏，公平对比。四测证实 HIO 反馈有得（support 演化）有失（尺度发散）。

### R4（新）. 配准是评估前提
相位恢复评估必须在平凡歧义群上配准后才有效（C5）。调研报告 §5.1/§5.2 已补。

### R5. 指标平等，amp_cc/ssim 都不可单独用（五测修正）
六个指标平等看待，分析时全看。amp_cc 在本 setup 和形态脱钩（C9）。**五测发现 ssim 也会失效**：HIO 收敛态（轮廓+周期斜线）ssim 被斜线拉到 0.017（假阴），关HM 平坦态 ssim 虚高 0.9（假阳）。多数情况 ssim 仍有参考价值，但 HIO 场景必看图。**以 real_space.png 为准（R8）。**

### R6（新）. 取末轮，不取最优瞬间
best_point / best_rho 改取末轮（收敛态），不取"ssim+amp_cc 最优瞬间"——最优瞬间是 cherry-pick，可能选到不稳定峰值（如实验3 it=340，之后发散）。

### R7（新）. CUDA 非确定性 → 多次跑取统计
单次结果随机（C6）。验证阶段可单次看趋势，**最后阶段必须多次跑取均值±方差**，结论才稳健。

### R8（新）. 看图为准
ssim/psnr 在"形态+周期斜线"和"值域漂移"场景都会失真。跑完必须先看 real_space.png 再下结论，不能用指标替代看图。

### R9（新）. 别用指标填补视觉空白
看不见图（工具限制）就明说、让用户描述，不要用指标"假读图"下结论。五测中段曾因此误判 HM 是元凶。

### R10（新）. 概率性收敛 → 多次短跑，而非单次长跑
HIO 这类收敛靠概率命中的算法，单次长跑不保证收敛。多次独立短跑（不同种子）+ 看图挑收敛，比赌单次长跑靠谱。

## 4. 指导后续

1. **跑新矩阵**（六测）：实验1·HIO 重复 3 次（撞收敛）+ 实验2/3 各 1200 轮，三实验都开 HM。跑完先看图（real_space_3runs.png 挑 HIO 收敛、实验2/3 看 real_space.png）。
2. **twin image 根因**（C11）：查 UNet+tanh+HIO 的上下半轮廓偏移。
3. **多次跑取统计**：最终结论需每实验 N 次取均值±方差，消除 C6 随机性。
4. **HIO 收敛客观判据**：目前靠人眼看"轮廓 vs 斜线"，探索能区分两者的指标（ssim 不行）。
