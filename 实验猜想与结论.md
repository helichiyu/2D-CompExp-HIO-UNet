# 实验猜想与结论

> 本文档沉淀 `迭代模型/对比` 项目实验至今的猜想与结论，**严格区分"已验证结论""待验证猜想""方法论反思"**。四测后大幅修订：引入配准评估（消除平移歧义）、发现 CUDA 非确定性、辨析 support 自锁与尺度发散。实验过程见 [初测汇报.md](初测汇报.md)、[二测汇报.md](二测汇报.md)、[四测汇报.md](四测汇报.md)。

> ⚠️ **全局前提（C6）**：UNet 实验受 CUDA 非确定性影响，**单次结果有随机性**。本文结论基于四测单次长测（HIO 10000 / UNet 3000 轮，配准评估，末轮取样），方向性可靠但绝对数值待多次跑取统计确认。

---

## 1. 已验证结论（有硬实验支撑）

### C1. 甲方 HIO 实现正确
真值 support 下漂亮收敛：ssim 0.95 / amp_cc 0.93 / IoU 0.77（初测）。严格 HIO 公式、FFT/IFFT、振幅替换、HM 实现无误。

### C2. HIO 动态 support 不收敛（四测 10000 轮重验坐实）
四测给 HIO 10000 轮（二测只 3000），配准后末轮：amp_cc 0.189、ssim 0.014、Δφ 1.569。与二测 3000 轮几乎一致——**十倍轮次仍不收敛**。根因是 support 估计的"鸡生蛋"循环依赖，不是轮次不够。

### C3. UNet 相位恢复（四测修正：配准后非"全未恢复"）
配准后实验3（tanh+HIO）在 it=340 达 **ssim 0.944**（形态恢复峰值，配准后真实值，与人眼看图一致，非暗背景虚高）。**配准前 amp_cc≈0 / Δφ=π/2 的"四版本全未恢复"判断是平移歧义假象**——配准后露出部分恢复。但实验3 末轮尺度发散（见 C8），未稳住。

### C4. sigmoid + HIO 反馈塌缩到平凡解
sigmoid（恒正）+ HIO 反馈，2000 轮：amp_cc=0.000、IoU=0.000，psnr/ssim 飙高是暗背景虚高假象，网络输出塌成近零/平坦图。

### C5（新）. 配准是评估前提（消除平凡歧义）
相位恢复有固有的**平凡歧义**——整体平移、共轭反转（twin image）、全局相位因子，三者傅里叶振幅完全等价。未配准时：Δφ 恒为 π/2、amp_cc→0 是**物体平移的特征签名**，与"完全没恢复"无法区分。四测配准后 amp_cc 从 ~0.09 升到 ~0.25，露出真实恢复。**不配准的指标无意义**。配准方法（FFT 互相关 + 歧义群枚举）经 sanity check 验证可靠（构造已知变换，配准内积 1.000）。来源见 [调研报告.md](调研报告.md) §5.1/§5.2 + [28-30]。

### C6（新）. CUDA 非确定性，实验不可复现
同代码、同 seed（UNet seed=0、相位 seed=42）、同机器，UNet 实验跑两遍 loss 从 it=20 就分叉（2.2138e-5 vs 2.1819e-5）。根因是 cuDNN 卷积/反传的非确定算法 + DIP 对初始轨迹的敏感。**迭代流程未变（git diff 客观确认），差异完全来自 GPU 非确定性**。这动摇了初测/二测所有单次 UNet 结论的稳健性。

### C7（新）. support 自锁（实验2 sigmoid+置0 的病根）
实验2 support 外置 0（`rho_next = raw_d × support`）→ `shrinkwrap(rho_next)` 永远估回原位 → **support 被初始轮廓焊死**，网络只在里面拟合振幅，形态无法演化。末轮 ssim 0.101（配准后），看图无轮廓；amp_cc 0.30 是振幅拟合假象（见 C9）。

### C8（新）. tanh+HIO 尺度发散（实验3 末轮）
实验3（tanh+HIO）末轮 psnr 走到 −38（rho 数值爆炸）。机制：① HIO 反馈是累加更新 `current_input − β·raw`，raw 有持续偏置则单调发散；② support 外无约束（HM 只在 support 内，置0 才有界，实验2 置0 不发散正是因此）；③ tanh 只限制网络输出 raw∈[−1,1]，不限制迭代状态 rho_next；④ 振幅 loss 只约束 FFT(raw×support) 振幅，不约束 rho_next 尺度。**loss 一直降、amp_cc 归一化平稳，但 rho_next 早已发散**。

**解法（已实施 relaxed HIO）**：support 外改 `γ·ρ−β·ρ′`（γ<1），把累加变成几何级数、收敛到有限稳态 `−βr/(1−γ)`，防发散（见 [调研报告.md](调研报告.md) §2.3、[31][32]）。受 RAAR（Luke 2005）松弛思路启发的工程简化，非严格 RAAR。实验1/3 同用，γ=0.9。待重跑验证 psnr 是否还走负。

### C9（新）. amp_cc 在本 setup 不可靠
amp_cc 量的是"振幅谱拟合度"，和形态恢复脱钩：实验2 amp_cc 0.30（loss 直接优化振幅，形态却锁死）；纯随机图配准后 amp_cc 也能到 0.2（基线）；实验3 amp_cc 0.15（低于基线，尺度发散扰动频域）。**amp_cc 在 0.15–0.30 区间横向比较三个实验基本无区分力**。

## 2. 核心猜想（四测后状态）

### H1. 纯振幅 loss 锁不住相位（修正）
- **原表述**（过于绝对）：相位完全锁不住，Δφ 卡 π/2 = 相位随机。
- **四测修正**：纯振幅 loss **能恢复低频形态**（实验3 it=340 配准后 ssim 0.944），但**锁不住高频相位**（Δφ 全程 π/2，因弱振幅频率无优化动力）。且 support 策略（自锁 vs 演化）是关键变量，不止 loss 本身。
- **状态**：部分成立，表述需细化。

### H2. HIO 反馈要求 ρ′ 振荡（实验3 支撑）
HIO 反馈 `ρ_k−β·ρ′` 要靠 support 外正负振荡抵消才不发散；tanh 输出虽可负但网络无动力振荡 → 累加发散。实验3 尺度发散（C8）支撑此猜想。

### H3. 网络输出不天然振荡
网络先验偏向平滑自然图，support 外不振荡；即使 tanh 允许负，也无动力画振荡。实验3 支撑。

### H4. 软学习 + HIO 反馈不兼容（部分推翻）
- **原表述**：HIO 反馈在软学习里有害。
- **四测修正**：HIO 反馈**让 support 能演化**（实验3 it=340 形态恢复，优于实验2 自锁），并非完全有害；但**引入尺度失控**（C8）。应改为"HIO 反馈打破 support 自锁、让形态能长，但代价是尺度发散"。
- **状态**：部分推翻，需重新表述。

## 3. 方法论反思

### R1. 前期 UNet 版本对比不严格
不同版本跑不同轮次、同时改多个变量。四测在配准 + 同 wall-clock（HIO 10000 / UNet 3000，各 ~17 min）下重新对比，才公平。

### R2. 区分"猜想"与"结论"
前期把"Δφ=π/2 即锁不住""HIO 反馈有害"当结论，实为猜想，被四测（配准后）部分推翻。今后严格区分。

### R3. "是否用 HIO 反馈"作为实验变量
不预设好坏，公平对比。四测证实 HIO 反馈有得（support 演化）有失（尺度发散）。

### R4（新）. 配准是评估前提
相位恢复评估必须在平凡歧义群上配准后才有效（C5）。调研报告 §5.1/§5.2 已补。

### R5（新）. 指标平等，amp_cc 不可单独用
六个指标平等看待，分析时全看。amp_cc 在本 setup 和形态脱钩（C9），不可单独作判据。配准后 ssim 最贴近形态（和人眼一致），可作形态代理，但仍需结合看图。

### R6（新）. 取末轮，不取最优瞬间
best_point / best_rho 改取末轮（收敛态），不取"ssim+amp_cc 最优瞬间"——最优瞬间是 cherry-pick，可能选到不稳定峰值（如实验3 it=340，之后发散）。

### R7（新）. CUDA 非确定性 → 多次跑取统计
单次结果随机（C6）。验证阶段可单次看趋势，**最后阶段必须多次跑取均值±方差**，结论才稳健。

## 4. 指导后续

1. **末轮形态验证**：best_rho 已改取末轮，重跑一次出末轮图，用眼睛 + 指标互相印证"末轮指标低到底是形态差、还是指标问题"。
2. **多次跑取统计**：最后阶段每实验跑 N 次，报指标均值±方差，消除 C6 的随机性。
3. **support 自锁的解法**：实验3 的 HIO 反馈是一种（让 support 演化），但代价是尺度发散；需找既能演化又不发散的 support 策略。
4. **尺度控制**：tanh+HIO 发散需解（如 support 外也截断/置0、或 loss 加 rho_next 尺度约束）。
5. **指标体系**：六指标平等，可考虑补"配准后实空间 Pearson CC"等更贴形态的指标（Pearson CC 是 [调研报告.md](调研报告.md) §5 已收录的标准指标，非新发明）。
