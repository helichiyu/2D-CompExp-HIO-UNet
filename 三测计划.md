# 三测计划

> 本文档规划 `迭代模型/对比` 项目的第三次测试。基于 [实验猜想与结论.md](实验猜想与结论.md) 的认识，在**严格控制变量**下对比三种方法，验证前期猜想。理论见 [调研报告.md](调研报告.md)，历史实验见 [初测汇报.md](初测汇报.md)、[二测汇报.md](二测汇报.md)。

---

## 1. 背景与动机

初测、二测及若干探测后，项目形成若干猜想（见 [实验猜想与结论.md](实验猜想与结论.md)），核心两条：
- **H1**：纯振幅 loss 锁不住相位（UNet 四版本 Δφ 全卡 π/2）；
- **H4**：软学习 + HIO 反馈不兼容（HIO 反馈在乙方无效甚至有害）。

但这些猜想的实验支撑**不严格**：不同 UNet 版本跑的轮次不同（5000 vs 2000），且每版改动多个变量（R1）。因此"是否使用 HIO 反馈"尚未在公平条件下验证。三测的目标：**把三种方法放在同一条件下一较高下，让数据说话。**

---

## 2. 实验目标

1. 在同图、同初始条件、算力相当的轮次下，对比三种方法的相位恢复效果；
2. 验证 **H1**：纯振幅 loss 的 UNet（实验 2/3）相位是否真的锁不住；
3. 验证 **H4**：乙方使用 HIO 反馈（实验 3）是否真的比不用（实验 2）差；
4. 重新验证 **C2**：甲方 HIO 给足 30000 轮（算力与 UNet 5000 轮相当）是否仍不收敛。

---

## 3. 三种实验定义

三种方法**共享实空间约束类型**（support + 正值 + 直方图匹配 + shrinkwrap），差异在像空间处理、输出层、support 外实空间策略。

### 实验 1：传统 HIO（甲方基线）
- **像空间**：FFT(ρ_k) → 振幅硬替换 |A_orig| → IFFT → ρ′（振幅**精确**满足）
- **实空间 support 外**：HIO 反馈 `ρ_k − β·ρ′`（β=0.7）
- **输出层**：无（非网络，直接迭代）
- **轮次**：30000（每轮 ~0.04s，算力与 UNet 5000 轮相当）

### 实验 2：UNet + sigmoid + 置 0（无 HIO 反馈）
- **像空间**：raw = UNet(current_input)，**sigmoid [0,1]**；rho_c = raw×support；loss = MSE(FFT(rho_c) 振幅, |A_orig|)，反传更新网络
- **实空间 support 外**：置 0（rho_next = HM(rho_c)，support 外随 rho_c=0）
- **轮次**：5000

### 实验 3：UNet + tanh + HIO 反馈
- **像空间**：raw = UNet(current_input)，**tanh [-1,1]**；rho_c = raw×support；loss = MSE(FFT(rho_c) 振幅, |A_orig|)，反传
- **实空间 support 外**：HIO 反馈 `current_input − β·raw`（β=0.7）+ HM
- **轮次**：5000

> **实验 2 vs 实验 3 的差异**：输出层（sigmoid vs tanh）+ support 外实空间（置 0 vs HIO 反馈），振幅 loss 对象一致（都用 raw×support）。两者是"自洽配对"（sigmoid 配置 0、tanh 配 HIO，因 sigmoid+HIO 已知塌缩，不自然）。若后续要严格分离单变量，可补 sigmoid+HIO、tanh+置 0 两个组合。

## 5. 实现方案（代码，待批准后实施）

1. **run_unet 参数化**：加模式开关，支持实验 2（sigmoid+置0）与实验 3（tanh+HIO）两种配置：
   - 输出层激活可选（sigmoid / tanh）——通过 `OutConv` 的激活参数控制；
   - support 外实空间策略可选（置 0 / HIO 反馈）——通过 `run_unet` 的参数控制；
   - 调用方按实验编号传模式，避免维护两份代码。
2. **main.py 改三实验编排**：依次跑 实验 1（HIO 30000）→ 实验 2（UNet sigmoid+置0 5000）→ 实验 3（UNet tanh+HIO 5000），结果分别存 `results/run_<时间戳>/exp1_hio/`、`exp2_unet_sigmoid/`、`exp3_unet_tanh/`。
3. **可视化**：复用二测的全套（实空间+误差、振幅谱、support、收敛曲线、指标表），每个实验独立一套；另加**三实验横向对比图/表**（amp_cc、Δφ、IoU 三柱状图 + 汇总表）。
4. 超参：HIO_ITER=30000（eval_every=100）、UNET_ITER=5000，其余不变。

## 6. 评估指标与判据

- **主指标**（判断相位是否真正恢复）：**amp_cc**（振幅域相关）、**Δφ**（平均相位误差）。真收敛标准：amp_cc > 0.5、Δφ < 1.0。
- **辅助指标**：ssim、psnr、IoU、pearson_cc。**注意暗背景虚高**——ssim/psnr 在偏暗图上可能虚高，不能单独作判据（C4 已示）。
- **收敛曲线**：看 amp_cc / Δφ 是否有离开初始值的趋势（而非仅 loss 下降）。

## 7. 预期结果与解读

| 实验 | 预期（基于猜想） | 解读 |
|---|---|---|
| 1 传统 HIO | amp_cc≈0、Δφ≈π/2（不收敛） | 坐实 C2：即使 30000 轮，动态 support 下 HIO 仍不收敛 |
| 2 UNet+sigmoid+置0 | Δφ≈π/2、amp_cc 或爬到 ~0.2 | 支撑 H1（相位锁不住）；amp_cc 来自振幅拟合，非相位恢复 |
| 3 UNet+tanh+HIO | Δφ≈π/2、amp_cc ≤ 实验 2 | 支撑 H4（HIO 反馈在软学习里无效/有害） |

**关键反例可能性**：
- 若实验 3 的 amp_cc **反超**实验 2 → 推翻 H4，说明 HIO 反馈在足够轮次下对软学习有益；
- 若三实验 Δφ **都卡 π/2** → 强支撑 H1：纯振幅 loss 框架的根本局限，要恢复相位必须跳出该框架（如纯 DIP 固定噪声输入，但需放宽"实空间逐字一致"原则）。
