# 实现方案：HIO vs 未训练 UNet 相位恢复对比实验

> 本文档是 `迭代模型/对比` 项目的工程实现方案，是写代码的直接依据。理论背景见 [调研报告.md](调研报告.md)。

---

## 1. 实验目标与控制变量设计

在同一张图 567.png（302×600，白背景 + 黑球分子结构）上，对比两种相位恢复方法：

- **甲方 = 传统严格 HIO 算法**（像空间：硬替换振幅）
- **乙方 = 未训练 UNet**（像空间：振幅 MSE 反传更新网络）

**控制变量法**：甲乙双方唯一区别是**像空间处理**；**实空间约束完全一致**（正值 + 实数 + 直方图匹配 + shrinkwrap 支撑域），从而隔离出"像空间处理方式"这一变量的影响。

数据来源：567.png → 灰度 → FFT 取振幅 |A_orig|（丢弃相位）。双方都只从 |A_orig| 出发恢复相位。

---

## 2. 关键技术决策

### 2.1 极性翻转（暗背景表示）—— 保证约束方向正确

567.png 是"白背景 + 黑物体"。直接归一化（白=1.0、黑≈0.1）会让所有实空间约束方向反向（shrinkwrap 把白背景误判为物体、support 外置 0 会清零白背景引入伪影、HM 参考峰落在背景上）。

**解法**：内部统一用**暗背景表示** `ρ_work = bg_val − img_01`（物体变亮、背景≈0），显示与存图时再 `bg_val − ρ_work` 翻回视觉原貌。

**控制变量不被破坏的数学依据**：
$$\mathcal{F}\{c - f\} = c\cdot N\cdot\delta_{(0,0)} - \mathcal{F}\{f\}$$
项目里直流分量 `fft[0,0]=0` 本来就置零，故**所有非直流频率的振幅谱完全相同** → |A_orig| 一字不变，甲乙双方像空间约束完全等价。

### 2.2 控制变量严格成立的 3 个对齐动作

1. **极性统一**：甲乙都用暗背景表示 `ρ_work`。
2. **约束施加点统一**：都在"当前密度估计"上施加——甲方在 IFFT 后的 ρ′_k 上，乙方在 UNet 输出后（与甲方对齐，而非仅输入前）。
3. **共享初始条件**：同一份随机相位种子（seed=42）→ 同一 ρ_0、同一初始 support、同一参考直方图。UNet 权重种子单独记录（这是方法内生差异，不破坏控制变量）。

### 2.3 数值范围约定

全程归一化到 [0,1]（暗背景下背景≈0、物体峰≈0.8–1.0）。FFT 振幅用 `amp/(H·W)` 归一化（仅用于 loss/显示），IFFT 配套 `/(H·W)`。所有指标 data_range=1.0。

## 4. utils.py 函数级设计

### `pad_to_multiple(x, m=32)` / `unpad(x, pad_info)`
pad 到 m 的倍数（UNet 5 级下采样需要），用 `F.pad(..., mode='constant', value=0)`（暗背景下 0=背景）。`pad_info=(left,right,top,bottom)`。

### `load_and_preprocess(path, expand=2, m=32, seed=42)`
返回 `(img_01, rho_work, bg_val, pad_info, H0, W0)`，均为 `[1,1,H,W]`。
```
gray = PIL(path).convert('L') → float32
bg_val_255 = Counter(gray).most_common(1)[0][0]          # 估背景(轮廓/main.py 技巧)
扩 expand 倍画布，居中填 bg_val_255                       # 缓解 FFT 卷绕(轮廓/main.py)
img_01 = canvas / 255.0
bg_val = bg_val_255 / 255.0
rho_work = bg_val - img_01                               # 极性翻转 → 暗背景
pad 到 m 倍数
```

### `fft_amp_phase(x)` → `(amp_raw, phase)`（可微）
```
F = torch.fft.fft2(x); F[...,0,0] = 0                    # 去直流(轮廓/main.py)
return torch.abs(F), torch.angle(F)
```

### `ifft_real(amp, phase)` → `rho`（可微）
```
rho = torch.real(torch.fft.ifft2(amp * exp(1j*phase))) / (H*W)
```

### `make_random_phase(shape, seed)` → `phase`
**实噪声 FFT 取相位**（保证共轭对称 → IFFT 为实数）：
```
torch.manual_seed(seed)
noise = torch.randn(shape)
F = fft2(noise); F[0,0] = 0
return angle(F)
```
比 `验证/custom.py` 的手工拼接共轭更稳。**甲乙共享 seed → 同一 ρ_0**。

### `init_density(amp_orig, phase_init)` → `rho_0`
`return ifft_real(amp_orig, phase_init)`

### `gaussian_kernel_2d(sigma, device)` → `[1,1,ks,ks]`（参考 custom.py L120-125）
`ks = 2*ceil(3*sigma)+1`；`x = arange(ks) - (ks-1)/2`；`exp(-(xx²+yy²)/(2σ²))`；归一化。

### `shrinkwrap_support(rho, sigma, thresh_frac=0.15)` → `mask {0,1}`
暗背景下物体 = 亮区，阈值方向"大于"：
```
blurred = F.conv2d(rho, gaussian_kernel_2d(sigma), padding='same')
thr = thresh_frac * blurred.max()
mask = (blurred > thr).float()
# 防 degenerate：mask 占比过高/过低自动调 thr
```

### `sigma_schedule(it, sigma0, sigma_end, total)` → `sigma`
每 `interval` 步用一次，线性衰减 `sigma0→sigma_end`。

### `estimate_reference_histogram(img_ref, support, n_bins=300)` → `(ref_edges, ref_means)`
**在 support 内**统计暗背景原图的密度直方图，**等数量分箱**（每 bin 像素数相同），返回 n_bins+1 个边界 + 每 bin 均值。

### `histogram_match(rho, support, ref_edges, ref_means, n_bins=300)` → `rho_matched`（`@torch.no_grad()`，Zhang-Main a/b）
```
当前密度在 support 内按自己的等数量边界分 bin → cur_edges
对每个 bin i：a_i=(r_hi-r_lo)/(c_hi-c_lo)，b_i=r_lo-a_i*c_lo
  （c_hi-c_lo<1e-8 时 identity，防除零）
support 内像素按所属 bin 做 a_i*ρ+b_i 线性映射；support 外不动
末尾 clamp 到 [0, bg_val]
```

### 指标函数（暗背景表示下，gt = rho_work）
- `psnr(a,b)`：`-10*log10(mean((a-b)²)+1e-12)`，MAX=1
- `ssim(a,b)`：`skimage.metrics.structural_similarity(a,b,data_range=1.0)`
- `pearson_cc(a,b)`：`(a-ā)·(b-b̄).sum() / (‖a-ā‖‖b-b̄‖+1e-12)`
- `amplitude_cc(ae,pe,ac,pc)`：搬 custom.py `calculate_CC_torch`：`Σ(ae·ac·cosΔφ)/√(Σae²·Σac²)`
- `mean_phase_error(pe,pc)`：搬 custom.py `calculate_DeltaPhi_torch`：`mean(arccos(cos(Δφ)))`
- `support_iou(mask_a, mask_b)`：`intersection/union`
- `evaluate_all(rho, gt, amp_orig, support_gt)` → dict

## 5. hio.py（甲方）

### `run_hio(amp_orig, rho_init, ref_info, gt, support_gt, max_iter=3000, beta=0.7, sigma0=3.0, sigma_end=1.0, sigma_interval=20, sigma_total=200, eval_every=20)` → `(best_rho, history)`

严格 HIO 一轮：
```
ρ_k → FFT → (|A_k|, φ_k)
像空间：F' = amp_orig * exp(1j*φ_k)            # 硬替换振幅，保留新相位
IFFT → ρ'_k
[每 sigma_interval 步] 用 ρ_k 更新 support (shrinkwrap, σ 衰减)
HIO 公式：
  keep = (support>0.5) & (ρ'_k >= 0)
  ρ_{k+1} = where(keep, ρ'_k, ρ_k - beta*ρ'_k)
HM：histogram_match(ρ_{k+1}, support, ref_info)   # 只在 support 内
评估（每 eval_every 步）：evaluate_all(ρ_{k+1}, gt, ...)
ρ_k = ρ_{k+1}
```
保留综合指标最优的 ρ 作为 best_rho。

**ρ_k / ρ′_k 关系**：ρ_k = 上轮输出（输入）；ρ′_k = 本轮傅里叶投影候选；ρ_{k+1} = HIO 公式组合。`keep→ρ′_k`；违反→`ρ_k−β·ρ′_k`。

## 6. unet_pr.py（乙方）

### `class UNet(nn.Module)`（自己重写，参考 network.py bilinear 版）
- `DoubleConv`：Conv3×3→**InstanceNorm2d**→ReLU→Conv3×3→InstanceNorm→ReLU（单图训练不用 BatchNorm）
- `Down`：MaxPool2d(2)→DoubleConv
- `Up`：bilinear Upsample(2) + concat 跳连 + DoubleConv，`F.pad` 处理奇偶差
- `OutConv`：Conv1×1 + Sigmoid，输出 **[0,1]**
- 5 级下采样，瓶颈通道 512（bilinear 时 factor=2 减半）

### `run_unet(amp_orig, rho_init, ref_info, gt, support_gt, max_iter=5000, lr=1e-4, sigma0=3.0, sigma_end=1.0, sigma_interval=20, sigma_total=500, eval_every=20, unet_seed=0)` → `(best_rho, history)`

DIP 一轮：
```
current_input = rho_init                    # 第一轮"先实空间"起手
opt = Adam(model, lr); amp_orig_norm = amp_orig/(H*W)
for it:
  opt.zero_grad()
  raw = model(current_input)                # sigmoid [0,1]
  # 实空间约束（可微部分，进 loss 路径，与甲方对齐）
  rho_c = raw.clamp_min(0) * support        # 正值 + support 外置0
  # 像空间：FFT 取振幅 → MSE → 反传（唯一变量）
  amp_pred = fft_amp_phase(rho_c)[0] / (H*W)
  loss = MSE(amp_pred, amp_orig_norm)
  loss.backward(); opt.step()
  # 实空间约束（非可微 HM，detach 喂回）
  with no_grad:
    rho_next = histogram_match(rho_c.detach(), support, ref_info)
    [每 sigma_interval 步] 更新 support
    评估、记录 best
    current_input = rho_next.detach()       # out.detach() 喂回(轮廓/main.py)
```

## 7. main.py 流程

```
1. img_01, rho_work, bg_val, pad_info, H0, W0 = load_and_preprocess('567.png')
2. amp_orig, _ = fft_amp_phase(rho_work)
3. phase0 = make_random_phase(shape, seed=42)          # 甲乙共享
4. rho_init = init_density(amp_orig, phase0)
5. support_gt = shrinkwrap_support(rho_work, sigma0)   # 评估用"真值 support"
6. ref_info = estimate_reference_histogram(rho_work, support_gt, n_bins=300)
7. best_hio,  hist_hio  = run_hio(amp_orig, rho_init, ref_info, rho_work, support_gt, ...)
8. best_unet, hist_unet = run_unet(amp_orig, rho_init, ref_info, rho_work, support_gt, ...)
9. 翻回视觉极性：vis = bg_val - rho；unpad
10. 出图：三连图(原图/HIO/UNet) + 收敛曲线(各指标 vs 迭代) + 指标表(CSV)
11. 存图存表
```
子函数：`plot_triple`、`plot_convergence`、`save_metrics_table`（matplotlib，SimHei 中文）。

## 9. 潜在坑清单

1. **FFT 卷绕** → 扩 2 倍画布
2. **直流分量** → `fft[0,0]=0`
3. **共轭对称** → 实噪声 FFT 取相位初始化
4. **HM 分箱除零** → `c_hi−c_lo<1e-8` 时 identity
5. **shrinkwrap mask 全 0/全 1** → 自动调 thr；σ0=3 较平滑避免碎裂
6. **UNet 输出范围** → sigmoid 约束 [0,1]；HM 后 clamp
7. **相位卷绕** → `arccos(cos)` 防卷绕
8. **扩边背景对 HM 的影响** → 参考直方图**必须在 support 内统计**
9. **尺寸非 32 倍数** → 600×2=1200 → pad 到 1216；302×2=604 → 608
10. **极性搞反** → 统一暗背景表示
11. **HIO 振幅尺度** → `amp_orig` 与 `phase_k` 同尺度，IFFT 的 `/(H·W)` 对称
12. **BatchNorm 单图训练** → 改 InstanceNorm2d

## 10. 验证方法

1. **单元自检**：`fft_amp_phase` → `ifft_real` 往返应能复原原图（误差 <1e-5）；`make_random_phase` 的 IFFT 虚部应近似 0。
2. **极性自检**：`fft_amp_phase(rho_work)` 与 `fft_amp_phase(img_01)` 的非直流振幅应完全相同。
3. **甲方收敛**：HIO 的振幅 CC 应随迭代上升，相位误差下降；best_rho 视觉上应重现黑球轮廓。
4. **乙方收敛**：UNet 的振幅 MSE loss 应下降，输出逐步成形。
5. **对比**：三连图肉眼对比 + 指标表定量对比（PSNR/SSIM/CC/Δφ/IoU）。
6. 运行：`D:\anaconda3\envs\use\python.exe main.py`。
