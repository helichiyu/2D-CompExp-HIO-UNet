## 4. 关键认知（六测核心）

### 4.1 三种方法的恢复都是概率性事件（★ 最重要的认知升级）
此前 C10 只把 **HIO** 收敛判为概率性。六测把"概率性"扩展到三种方法：
- **HIO**（C10 已知）：四测/五测/六测多次都没收敛（概率命中）；
- **UNet sigmoid**（C7 + 概率）：四测 3000 轮没触发相变、五测 2000 轮触发、六测 1200 轮触发——能否相变高度随机；
- **UNet tanh+HIO**（六测新证）：五测 2000 轮 ssim 0.93 形态在，**六测 1200 轮没恢复**——同代码不同次结果迥异。

> 方向性可靠（跨测一致），绝对数值/单次结论非稳健（C6 CUDA 非确定性）。**最终结论必须多次跑取命中率/均值±方差**（用户核心建议）。

### 4.2 HIO 非收敛的"中间态"（细化 C10）
五测 C10 说"不收敛 = 周期斜线"。六测看到另一种非收敛态：**中心有轮廓（部分恢复）+ 周边周期性"轮廓循环"**（分子结构感），中心最明显。说明 HIO 在往收敛走（中心出了形态），但没完全收敛。这与 C10「收敛是概率命中、长迭代无益」存在张力——**可能存在"部分收敛 → 更完全收敛"的路径，待加轮次验证**（见 §5.1）。

### 4.3 实验2·tanh+HIO 单次未恢复 + 十字星伪影（C11 旁）
六测 tanh+HIO 没恢复（看图确认），且**未复现五测的典型 twin image**（C11），取而代之是**边缘高亮 + 配准"十字星"伪影**：疑似 rho 边界值域漂移（psnr −5.63）+ FFT 互相关配准受边界强值影响产生十字形。ssim 0.819 是假阳（R5 再现）。relaxed HIO（γ=0.9）防住了 −38 爆炸（C8 仍立），但末轮尺度漂移方差大（五测 +4.85 / 六测 −5.63）。

### 4.4 实验3·sigmoid 1200 轮内相变（C7 强复现）
全程：0 轮 ssim 0.518 → 580 轮 0.588（徘徊）→ 960 轮 0.865（相变起点）→ 1180 轮 **0.952**。**比五测（2000 轮、780–980 轮相变）更早触发**，C7「非死锁、是慢收敛相变」更稳。amp_cc 0.414（三实验最高，且这次与形态一致）。

## 5. 未定与后续

1. **HIO 增加轮次**（用户建议）：六测看到"中心部分恢复"中间态，加轮次验证能否完全收敛；同时审视 C10「长迭代无益」是否需要修正。
2. **多次重复取统计**（用户核心建议）：三种方法都是概率恢复，每实验 N 次取命中率/均值±方差，消除 C6 随机性，得稳健结论。
3. **twin image / 十字星**（C11 延续）：六测 tanh+HIO 未复现典型 twin image、出现十字星伪影，待多次跑后看清这是否是稳定现象。
4. **HIO 收敛客观判据**（五测遗留）：仍靠人眼，本次更复杂（轮廓循环 vs 斜线 vs 收敛），客观指标更急需。
