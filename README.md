# 对比实验：传统 HIO vs 未训练 UNet 相位恢复

> **五测（2026-07）**：三实验均开 HM、各 1200 轮；实验1·HIO 重复 3 次（收敛概率性，多次撞收敛）。关键修订与发现见 [五测汇报.md](五测汇报.md)、[实验猜想与结论.md](实验猜想与结论.md)。

在同一张图（567.png）上，严格控制变量对比**三种**二维相位恢复方法（均开直方图匹配 HM、各 1200 轮）：

- **实验1·HIO**：传统严格 HIO（像空间硬替换振幅；relaxed HIO γ=0.9 防尺度爆炸）——**重复 3 次**（不同相位种子；HIO 收敛是概率性事件，多次撞收敛，出 3 次并排图供看图挑选）
- **实验2·tanh+HIO**：未训练 UNet + tanh + HIO 反馈（UNet 输出做 FFT 取振幅，与 |A_orig| 算 MSE 反传；support 外用 HIO 反馈）
- **实验3·sigmoid+置0**：未训练 UNet + sigmoid + 置 0（同实验2 像空间；support 外置 0）

三实验**实空间约束完全一致**（正值 + 实数 + HM + shrinkwrap），差异仅在像空间处理、support 外策略、输出层。**评估前配准**（消除平移/共轭反转/符号的平凡歧义）。

## 背景

X 射线晶体学 / 相干衍射成像中，探测器只能测得傅里叶振幅 |F|，**相位丢失**（相位问题）。本实验从 567.png 的傅里叶振幅出发，丢掉相位，分别用三种方法恢复。

## 测试建议（重要）

完整迭代耗时长，**强烈建议先用少轮次验证能跑通**，再放量：

1. **短测**：`main.py 20 10` 验证三实验跑通 + 全套出图；
2. **缓慢加轮次**看趋势（200 → 600 → 1200）；
3. 最后跑完整 `main.py` 出最终结果。

> 相位恢复 / 未训练网络对超参敏感，少轮次试探能大幅节省时间。四测发现 UNet 实验还有 **CUDA 非确定性**（同代码同 seed 跑两遍结果不同），单次结果有随机性，最后阶段需多次跑取统计。

## 关键设计

1. **极性翻转（暗背景表示）**：内部统一 `ρ_work = bg_val − img`（物体亮、背景≈0）。`FFT(c−f)` 与 `FFT(f)` 非直流振幅相同，振幅约束不变，控制变量不破。

2. **保留直流**：相位恢复需要平均密度信息，去直流会导致不收敛。

3. **三实验像空间差异**（唯一变量）：实验1 硬替换振幅；实验2/3 UNet 输出做 FFT 取振幅、与 |A_orig| 算 MSE 反传。

4. **输出层参数化**：`OutConv` 激活可选——`sigmoid`（[0,1] 恒正，配 support 外置0）/ `tanh`（[−1,1] 可负，配 HIO 反馈）。

5. **support 外策略**：置 0（实验2，天然有界）/ **relaxed HIO** `γ·ρ−β·ρ′`（实验1/3，γ<1 防累加发散，γ=0.9）。

6. **shrinkwrap 动态支撑域**：高斯模糊 + 阈值二值化估物体区域，不偷看原图。σ 线性衰减（3.0→1.0），周期更新。

7. **直方图匹配**（Zhang-Main 300 等数量 bin）：参考直方图从原图统计，只在 support 内做。

8. **评估前配准**（★ 四测新增）：相位恢复有平凡歧义（平移/共轭反转/符号），未配准的 Δφ/amp_cc 被平移污染。`evaluate_all(align_to_gt=True)` 先在歧义群配准（FFT 互相关 + 翻转/符号枚举）再算指标。详见 [调研报告.md](调研报告.md) §5.1/§5.2。

9. **评估前投影**：ρ 投影到可行域 `clamp_min(0) × support_pred`，消除 support 外反馈值干扰。甲乙一致。

10. **取末轮**：`best_rho` / `best_point` 取末轮（收敛态），不取最优瞬间（cherry-pick，可能选不稳定峰值）。

11. **轮次与重复**：五测起统一 1200 轮（收敛是早期概率命中，长迭代无益甚至有害）；实验1·HIO 重复 3 次（收敛概率性，多次撞收敛，出并排图供看图挑选）。

## 评估指标（全套，配准后算）

| 指标 | 含义 | 反映 |
|---|---|---|
| PSNR | 峰值信噪比（实空间） | 像素误差（受尺度干扰） |
| SSIM | 结构相似（11×11 高斯窗） | **形态/结构（最贴人眼）** |
| Pearson CC | 线性相关（实空间） | 形态整体相关 |
| amp_cc | 振幅域相关 ΣF_o·F_c·cosΔφ/√(...) | 振幅谱拟合（可能和形态脱钩） |
| Δφ | 平均相位误差（arccos·cos 防卷绕） | 频域相位 |
| support IoU | 估 support 与真值的交并比 | support 位置 |

六指标**平等看待**，分析时全看。配准后 SSIM 多数情况贴形态，但 HIO"形态+周期斜线"场景会失真（R5），**必看 real_space.png**。详见 [调研报告.md](调研报告.md) §5。

## 五测状态（看图为准）

- **HIO 收敛概率性**（C2/C10）：收敛→轮廓、不收敛→周期斜线，与迭代长度无单调关系。长跑恰好没收敛（斜线）。对策：实验1 重复 3 次撞收敛。
- **UNet+sigmoid 慢相变收敛，C7 推翻**：长跑 2000 轮 ssim 0.95、看图好，780–980 轮相变。非 support 自锁死锁。
- **UNet+tanh+HIO relaxed HIO 修复尺度爆炸**（C8）：末轮 psnr 不再 −38，rho 不再爆炸。但仍有 twin image（C11，待查）。
- **HM 是双刃剑非元凶**：开 HM 带轮廓（也带斜线），关 HM 形态更差。三实验都保留 HM。
- **CUDA 非确定性**（C6）：UNet 不可复现（sigmoid 四测 ssim 0.10 / 五测 0.95），单次结果随机。
- **ssim 局限**（R5）：HIO"形态+周期斜线"场景 ssim 失真（假阴/假阳）。**以 real_space.png 看图为准。**

详见 [实验猜想与结论.md](实验猜想与结论.md)、[五测汇报.md](五测汇报.md)。后续：跑新矩阵、twin image 根因、多次跑取统计、HIO 收敛客观判据。
