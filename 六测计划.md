# 六测计划

> 本文档规划 `迭代模型/对比` 项目的第六次测试。五测末已重组 [main.py](main.py) 实验矩阵（三实验均开 HM、各 1200 轮、实验1·HIO 重复 3 次），但该新矩阵尚未实际跑过。六测目标：**跑一遍新矩阵，看图下结论（人眼为准，R8），写六测汇报，沉淀认知。不改代码**（不含 twin image 根因、多次统计、HIO 判据等扩展——见 §5 后续）。

> 理论见 [调研报告.md](调研报告.md)，五测认知见 [五测汇报.md](五测汇报.md)、[实验猜想与结论.md](实验猜想与结论.md)，历史见 [初测汇报.md](初测汇报.md)、[二测汇报.md](二测汇报.md)、[四测汇报.md](四测汇报.md)。

## 1. 背景与动机

五测核心认知（详见 [实验猜想与结论.md](实验猜想与结论.md)）：
- **HIO 收敛是概率性事件**（C2/C10）：收敛→轮廓、不收敛→周期斜线，与迭代长度无单调关系。对策：实验1 重复 3 次撞收敛。
- **relaxed HIO（γ=0.9）修复尺度爆炸**（C8）：实验2 末轮 psnr 不再走负。
- **sigmoid 慢相变收敛**（C7 推翻）：长跑 2000 轮 ssim 0.95，非死锁。
- **HM 是双刃剑非元凶**：三实验都保留 HM。
- **看图为准**（R8/R9）：ssim 在"形态+周期斜线"场景失真，跑完必看 real_space.png，不能用指标替代看图。

五测末据此重组 [main.py](main.py) 矩阵（[五测汇报.md](五测汇报.md) §6），但新矩阵未跑。六测即跑这个矩阵，用眼睛与指标互相印证，收尾本轮对比。

---

## 2. 实验目标

1. 跑通五测铺好的新矩阵（HIO×3 + UNet tanh+HIO + UNet sigmoid+置0，都开 HM、各 1200 轮）；
2. 看图判断：HIO 3 次哪几次收敛（轮廓 vs 周期斜线）；实验2/3 末轮形态如何；
3. 三实验末轮指标横向对比（配准后，取末轮）；
4. 据结果写六测汇报，更新 [实验猜想与结论.md](实验猜想与结论.md)。

## 4. 实施流程

### 阶段 1：全量跑（1200 轮）

### 阶段 2：看图下结论（R8/R9：人眼为准）
> 教训 R9：工具读图只回链接、视觉内容不进上下文，五测曾因此误判 HM 是元凶。**跑完由用户看图描述，我不用指标填补视觉空白。**

请用户重点看并描述：

结合指标（`comparison.csv` 末轮 + 各 `history.csv` 全程趋势）分析，但**以看图为准**。

### 阶段 3：写汇报 + 更新认知
