# 相位恢复调研报告：传统 HIO 与未训练神经网络方法

> 本报告服务于 `迭代模型/对比` 项目的对比实验设计：在同一张图上对比**传统严格 HIO 算法**与**未训练 UNet（deep image prior 路线）**两种相位恢复方法。报告梳理了相位恢复的数学框架、各类实空间约束、HIO 算法、直方图匹配、未训练神经网络方案与评估指标，并给出本项目最终采用的方案。所有结论均基于对本地文献库与已有代码的实际精读，参考文献见文末。

---

## 引言：相位问题

在 X 射线晶体学、相干衍射成像（CDI）等场景中，探测器只能测量散射波的**强度**（即傅里叶变换的振幅平方 |F|²），**相位信息丢失**。但从振幅反演实空间结构（电子密度 / 物体轮廓）却离不开相位——这就是晶体学著名的**相位问题**（phase problem）。

记实空间物体为 ρ(x)，其傅里叶变换为：

$$F(k) = \mathcal{F}\{\rho\}(k) = |F(k)|\,e^{i\varphi(k)}$$

实验只测得 |F(k)|，相位 φ(k) 未知。相位恢复（phase retrieval）就是要从 |F| 反推出能重建 ρ 的相位。这是一个高度病态的反问题：许多不同的相位组合能产生相同的振幅，解不唯一。要锁定物理上有意义的解，必须引入**先验约束**，在实空间与频率（像）空间之间交替投影迭代 [1,2]。

## 1. 相位恢复的数学框架：交替投影

相位恢复迭代算法的本质是**交替投影**（alternating projections）：在两个约束集之间反复投影。

- **像空间约束**（Fourier / modulus constraint）：恢复图的傅里叶振幅必须匹配实验测得的 |F_orig|。
- **实空间约束**（real-space constraint）：恢复图必须满足物理先验（如非负、有界支撑、特定密度分布等）。

每一轮迭代：
1. 实空间图 ρ → FFT → (|F|, φ)
2. **像空间投影**：用 |F_orig| 替换 |F|，保留 φ → F'
3. IFFT(F') → 实空间图 ρ′
4. **实空间投影**：对 ρ′ 施加先验约束 → ρ_next
5. 回到 1

不同算法（ER、HIO、DM、RAAR 等）的区别，就在于第 4 步"实空间投影"怎么做 [1]。

## 2. 传统迭代算法

### 2.1 误差下降法 ER（Error Reduction）

最简单的实空间投影：对违反约束的像素**直接裁剪**（置零或截断），满足约束的像素保留 [2]：

$$\rho_{k+1}(x) = \begin{cases} \rho'_k(x), & x \text{ 满足所有实空间约束} \\ 0, & \text{否则} \end{cases}$$

ER 稳定但易陷入停滞（stagnation）——一旦违反约束的像素被清零，下一轮可能又产生同样的违反，导致停滞在局部解。

### 2.2 混合输入输出算法 HIO（Fienup 1982）

HIO 是 Fienup 为**二维图像相位恢复**提出的算法，核心改进是用"反馈"代替"清零"，跳出停滞 [1]。

**严格 HIO 公式**：

$$\rho_{k+1}(x) = \begin{cases} \rho'_k(x), & x \in \text{support 且 } \rho'_k(x) \geq 0 \\ \rho_k(x) - \beta\,\rho'_k(x), & \text{否则} \end{cases}$$

其中：
- ρ'_k 是第 k 轮像空间投影后反变换回的实空间图（满足振幅约束）；
- ρ_k 是上一轮的实空间输出；
- β 是反馈系数，典型取 **0.5–1.0**，常用 0.7–0.9；
- "否则"包括两种违反：① 在支撑域外；② 支撑域内但密度为负。

HIO 的精髓：对违反约束的点不做硬清零，而是用 `ρ_k − β·ρ'_k` 做**反向修正**（hybrid），保留"惯性"，从而逃离 ER 的停滞。本项目实验1/3 采用 relaxed HIO 变种（见下文 §2.3），β=0.7、γ=0.9。

### 2.3 relaxed HIO（防 support 外累加发散，★ 四测后采用）

标准 HIO 的 support 外更新 `ρ_k − β·ρ′` 是**累加式**：每轮 `ρ_next[t] = ρ_next[t−1] − β·ρ′[t]`。若 ρ′ 在 support 外有持续偏置 r，ρ_next 会**线性发散**（四测实验3 tanh+HIO 末轮 psnr 走到 −38，正是此机制）。

RAAR / relaxed HIO 家族（[31] Bauschke 2002；[32] Luke 2005）通过松弛系数把累加稳定化。本项目采用其工程简化版——support 外加松弛系数 γ<1：

$$\rho_{next} = \gamma\,\rho_k - \beta\,\rho' \quad (\text{support 外};\ \gamma<1)$$

数学上 support 外 `ρ_next[t] = γ·ρ_next[t−1] − β·ρ′[t]`，若 ρ′ 稳定 r，收敛到有限稳态 $-\beta r/(1-\gamma)$，不再发散。γ=0.9 → 稳态 ≈ −7r；γ=0.5 → ≈ −1.4r。γ 越小越稳，但反馈力度也越弱（support 演化能力下降）。

本项目实验1（HIO）与实验3（UNet+HIO）均采用此 relaxed 公式（β=0.7、γ=0.9），保证两者 support 外策略一致、公平对比；实验2（置0）不涉及。

> 注：RAAR 原算子 $T_\beta = \tfrac{\beta}{2}(R_1 R_2 + I) + (1-\beta)P_2$（[32]）更复杂；本项目的 $\gamma\rho_k - \beta\rho'$ 是受其松弛思路启发的工程简化，侧重防 support 外累加发散，**非严格 RAAR**。

### 2.4 ER + HIO 交替（避免 stagnation / 周期条纹，★ 八测方向）

单纯 HIO（含 relaxed）在 Fourier 空间用负反馈逼近 support 约束，反馈参数（β/γ）不当时会**震荡 / 停滞（stagnation）**到周期态，在实空间表现为周期条纹 / 斜线 [1, 33]。ER 则稳定但易陷停滞。两者**交替运行**是标准做法——例如每 N 轮 HIO 后插入几轮 ER：ER 的硬投影收敛局部、HIO 的反馈逃离停滞。[33] §3.4 明确：「ER 与 HIO 常以交替方式组合使用，以避免迭代过程的停滞或震荡、稳定最终重建」。

> **周期斜线的两种可能机制**（[33] §3.6.2）：① **HIO 负反馈震荡 / stagnation**——本项目七测的主因（HIO / tanh+HIO 曲线始终震荡、sigmoid 无 HIO 反馈则丝滑无斜线）；② **FFT 卷绕效应（wrapping）**——物体谱无限而 FFT 谱有限（Nyquist），超出频率卷绕到谱边缘成实空间条纹，但仅"物体边缘信号强 + padding 不足"时显著，本项目物体居中、padding 充足，wrapping 影响小。故八测主攻 ①（ER+HIO 交替、多次平均）。

本项目实验1（纯 HIO）、实验2/3（纯 UNet）当前均未交替。

**八测查证**：Fienup 1982 经典是 ER→HIO→ER（ER 开头+收尾，原文 Fig 8：ER 20 → HIO 40 → ER 10）；Latychevskaia 2018 §3.4 提 alternating 但**自己 examples 用纯 HIO**（选误差最小 10 个对齐平均 + PRTF）；ER+HIO 交替在文献多用于**复杂相位物体**（Harder 2010 complex density、Newton 2009 strain），**实物体无精确配方**。八测试 20H+5E 周期穿插失败（ER 在 HIO 未建立物体时穿插、硬清零侵蚀物体），废弃。详见 [八测阶段汇报.md](八测阶段汇报.md) §5、[Fienup 1982 原文](https://labsites.rochester.edu/fienup/wp-content/uploads/2019/07/AO82_PRComparison.pdf)。

### 2.5 RAAR / Difference Map（现代算法，九测实施）

ER+HIO 交替是 1982 年老路线。现代 CDI 主流是 **RAAR** / **DM**（Hawk 等软件标配）。

**RAAR（Luke 2005，[32]）**——反射算子框架：

$$x_{n+1} = \tfrac{\beta}{2}\big(R_M(R_S(x_n)) + x_n\big) + (1-\beta)\,R_S(x_n), \qquad R = 2P - I$$

两个约束集（实空间 S = support+正值、像空间 M = 振幅）**都用反射 R=2P−I**（越过约束边界、跳局部最小）+ β 松弛平均，替换 HIO 的“投影+反馈”。反射无累加项 → 理论上避 HIO 负反馈震荡（C14）。§2.3 的 relaxed HIO（`γ·ρ−β·ρ′`）是其工程简化。**九测参数 β=0.7**：Luke 2005 给 β∈(0,1)，CDI 实践常用 0.7–0.9；取 0.7 与项目 HIO 的 β=0.7 一致好对照（详见 [九测计划.md](九测计划.md) §3.3）。

**Difference Map（Elser 2003）**——投影差框架：

$$D(x) = x + \beta\,\big[\pi_1\!\big(x+\gamma_2(\pi_2(x)-x)\big) - \pi_2\!\big(x+\gamma_1(\pi_1(x)-x)\big)\big]$$

**标准参数 β=1、γ₁=−1、γ₂=±1/β 时退化为 Douglas–Rachford ≡ Fienup HIO**——[31] Bauschke 2002 从凸优化视角证明 DM ≡ DR ≡ HIO，Fienup 2013《Phase retrieval algorithms: a personal tour》明确「DM has HIO as a special case」，[Wikipedia](https://en.wikipedia.org/wiki/Difference-map_algorithm) 同。即用 β=1 等于测 HIO，九测白做。⚠️ **γ₂ 符号文献有 ±1/β 两种写法**（投影顺序 / 符号约定差异），实现时以 [Elser 2003 原文](https://arxiv.org/abs/math/0111080) 公式形态为准，并用「β=1.1 时 DM 结果应明显不同于 HIO」反验符号正确性。

**九测参数 β=1.1**：Elser 本人推荐 β≈1.1–1.2 创造「非退化 wandering 不动点」以逃 stagnation，脱离 β=1 退化点（β≠1 即不严格退化）；γ 用 Elser 标准关系（详见 [九测计划.md](九测计划.md) §3.4）。

> 九测实施 RAAR（β=0.7）+ DM（β=1.1）替换 HIO 两槽位，验证反射 / 投影差框架能否破 C14 周期斜线（relaxed HIO 是 RAAR 工程简化、DM 标准 β=1 退化 HIO 故取非退化值）。来源：[Luke 2005 / arXiv math/0405208](https://arxiv.org/abs/math/0405208)、[Elser 2003 / arXiv math/0111080](https://arxiv.org/abs/math/0111080)、[Bauschke 2002 [31]](https://pcombet.math.ncsu.edu/josa1.pdf)、[Wikipedia: Difference-map algorithm](https://en.wikipedia.org/wiki/Difference-map_algorithm)。

**九测实测结论**：RAAR（β=0.7）5/5 稳定 ssim 0.945–0.951、曲线丝滑、背景干净——**C14 周期斜线被破解（C15）**，反射框架证实优于 HIO"投影+反馈"；DM（β=1.1）5/5 ssim 0.10，本 setup（support+positivity+HM）下不工作，**排除**。详见 [九测汇报.md](九测汇报.md)。

## 3. 实空间约束方法

实空间约束是相位恢复能否收敛的关键。下面逐一梳理文献中出现的约束算子，并标注对二维图像场景的适用性。

### 3.1 支撑域约束 Support

物体只存在于有限区域内，区域外密度为零 [1,12]：

$$\sigma(x) = \begin{cases} 1, & x \in D \\ 0, & x \notin D \end{cases}$$

D 即支撑域（support）。对"黑球 + 白背景"这类图，支撑域 = 物体（前景）所在区域，天然明确。支撑域是相位恢复**最强**的实空间先验之一，HIO 的反向修正规则就以它为分界。

支撑域的获取有两种：
- **静态 / 先验给定**：从其他途径（如低分辨率分子形状）预先得到——但真实场景往往没有；
- **动态估计**：算法在迭代中自己估，见 3.9 节 shrinkwrap。

### 3.2 正值约束 Positivity

电子密度物理上非负：ρ(x) ≥ 0 [15]。这是隐含的物理约束。HIO 公式中"支撑域内但 ρ′ < 0"也按违反约束处理，正是正值约束的体现。

### 3.3 实数约束 Reality

实空间密度是实数（虚部为零），即 IFFT 后只取实部。这是实信号相位恢复的默认操作，保证了频域的共轭对称性。

### 3.4 直方图匹配 Histogram Matching（★ 本项目重点）

**原理**：正确的电子密度图，其像素值的频率分布（直方图）具有特定形状。Lunin (1988) [5] 首先提出用密度分布作为约束；Zhang & Main (1990) [4] 将其简化为**分箱线性映射**，成为 CCP4 DM 程序 [8] 的标准做法。

**Zhang-Main 分箱线性映射算法** [4,24]：
1. 把当前密度图与参考（目标）密度图各自分成 **含等数量网格点的 bin**（典型 n ≈ 100–300）；
2. 当前图第 i 个 bin 边界为 [θᵢ, θᵢ₊₁]，参考图为 [θ′ᵢ, θ′ᵢ₊₁]；
3. 在每个 bin 内做线性映射 `ρ_new = a·ρ + b`：

$$a = \frac{\theta'_{i+1} - \theta'_i}{\theta_{i+1} - \theta_i}, \qquad b = \frac{\theta'_i\theta_{i+1} - \theta'_{i+1}\theta_i}{\theta_{i+1} - \theta_i}$$

4. 反变换得到改进的相位。

等价地，直方图匹配就是用参考累积分布把当前密度的 CDF 拉平：ρ_new = F_ref⁻¹(F_cur(ρ_cur))。

**作用区域（文献核实，2026-07）**：HM 只在**分子区（protein region）**做，溶剂区由 solvent flattening（§3.5）管——这是 Zhang & Main 1990 的原始分工 [4]，Cowtan 1999 [8]、CCP4 课程均明确（"applied to the protein region of a map"）。本项目 `histogram_match` 只在 support 内做（[utils.py:220](utils.py#L220) `cur_vals = rho[support>0.5]`），对应晶体学的 protein region，**忠实原始文献**；support 外（背景）由 positivity/support 置零约束管（solvent flattening 的特例，§3.5 注）。

**参考（目标）直方图从哪来**——这是核心问题：
- **蛋白质晶体学标准做法** [4,5,8,12]：取一个**已知结构的别的蛋白质 PDB 模型**（非当前待解对象），算它在**同分辨率**下的理想电子密度图，统计其密度直方图作为参考。依据是 Zhang & Main 的论点：**相同分辨率下，蛋白质电子密度直方图是普适的**，与具体是哪个蛋白无关，只由分辨率、氨基酸组成、溶剂含量决定。本地 C++ 代码 `computeRefeHist.cpp` 即从已知 3tqe 结构算出 `3tqe_hist.txt`（300 个 bin 边界）作为参考 [12]。
- 本项目 567.png 是单张普通图像，不存在"同类图像普适分布"。最接近上述晶体学做法的方案是**从原图统计参考直方图**（相当于"拿已知结构算参考"），甲乙双方共用同一份，保证控制变量公平。

**效果**（刘忠川博士论文表 4-2，蛋白质 3tqe 不同溶剂含量体系）[12]：

| 体系 | 溶剂含量 | 无 HM 的 CC | 有 HM 的 CC | 无 HM 相位误差(°) | 有 HM 相位误差(°) |
|---|---|---|---|---|---|
| P41-100 | 70.34% | 0.97 | 0.95 | 34.8 | 45.3 |
| P41-95 | 65.75% | 0.89 | 0.87 | 51.6 | 53.8 |
| P41-90 | 59.61% | 0.81 | 0.82 | 66.0 | 60.4 |
| **P41-85** | **52.25%** | **0.27** | **0.87** | 87.9 | **57.0** |
| P41-80 | 42.99% | 0.15 | 0.07 | 89.7 | 89.2 |

**结论**：直方图匹配对**临界过采样**（溶剂刚过 50%）体系是决定性的（CC 从 0.27 跃升到 0.87，相位误差从 88° 降到 57°）；对高溶剂/已收敛体系略有负面；对严重欠采样无力。它扮演的是"二次约束"角色，本地 C++ 代码里每 3 步迭代插入一次 [12]。

### 3.5 溶剂平摊 Solvent Flattening（Wang 1985）

把分子包络外的溶剂区电子密度强制设为低常数（通常取平均溶剂密度或 0）[3,12,24]。物理依据：溶剂区是均匀平坦的水溶剂，密度涨落远小于蛋白质内部，强制平坦化能压低噪声、提高信噪比。溶剂含量越高，效果越强。

Wang 的实空间方法：在每个网格点用以该点为中心、半径 R 的球内的加权平均替换该点密度，权重 `w = (1 − r/R)`（仅对正密度）[3]。Leslie (1987) 给出了等价的倒易空间（卷积定理）快速实现 [12]。

> 注：对二维图像，"背景区置零"的支撑域约束（3.1）已是溶剂平摊的特例，故本项目不再单列溶剂平摊。

### 3.6 溶剂反转 Solvent Flipping（Abrahams 1997）

solvent flattening 会引入偏置。Abrahams 改进：包络外不归零而**反号** `g(x) = −α`（α = V_蛋白/V_胞），消除原结构因子偏置，让修改后的数据与原数据统计独立 [6,24]。

### 3.7 非晶体学对称性平均 NCS Averaging（Bricogne 1974）

若晶胞内存在多个相同的分子拷贝（非晶体学对称，NCS），按对称操作把它们 的密度平均，可降噪、提高信噪比 √N 倍 [7,11]。**普通二维图像无对称性，不适用**。

### 3.8 多支撑域 Multi-support

把空间分为核心区（ER 约束）、纯溶剂区（置零）、过渡带（HIO 约束）三部分，分别施加不同约束，可进一步改善相位 [12]。

### 3.9 动态支撑域估计：Shrinkwrap（Marchesini 2003）

真实场景往往没有预知的支撑域。**shrinkwrap** 是 CDI 领域估计支撑域的标准方法（名字来自"收缩膜包装"）[9]：

每隔几轮迭代做一次：
1. 取当前恢复图 ρ（不完美，有噪声）；
2. 做**高斯模糊**（去毛刺、压噪声，让物体轮廓浮现）；
3. 取阈值（如模糊后最大值的 20% 或最亮像素的某分位）；
4. 模糊密度 ≥ 阈值 → 支撑域内（=1）；< 阈值 → 支撑域外（=0）；
5. 得到新的二值 mask，下一轮用；随迭代推进可让高斯 σ 递减，收紧边界。

**关键性质**：shrinkwrap 由算法**自己**估支撑域，不需要原图，属于"无先验"。几乎所有现代 HIO 都用它。本地 C++ 代码用等价的 Wang 平滑 + 阈值法动态生成蛋白质 mask [12]。

**参数取值（文献，十一测依据）**：
- **阈值**：典型 **0.1–0.2**（Marchesini 2003 原始 0.15–0.20 [9]；[Latychevskaia 2018](https://arxiv.org/pdf/1809.04626) 综述 0.1–0.2；[Yoshida 2024](https://pmc.ncbi.nlm.nih.gov/articles/PMC10833425/) 多粒子 aggregate 场景用 0.04）。**无普适值，依物体形态实证扫**；机理上 higher threshold → smaller reconstructed area（[Tadesse 博士论文](https://www.db-thueringen.de/servlets/MCRFileNodeServlet/dbt_derivate_00046585/dissertationtadesse.pdf)）。
- **高斯 σ**：~3 pixel（项目 sigma0=3.0），随迭代递减收紧边界。
- **更新周期**：每 N 轮（文献 50–100，如 Yoshida 每 100 HIO 轩；项目 sigma_interval=20 更频繁）。

**改进型 SW（[Yoshida 2024](https://pmc.ncbi.nlm.nih.gov/articles/PMC10833425/)）**：σ 不固定递减，而按 **support 面积变化幅度（|ΔOS ratio|）自适应**——剧变时大 σ=2（强平滑，抑制 support 突变导致 PR 发散）、稳定后切 σ=0.9（紧贴物体）。针对 support 剧变型发散；项目十测组4 unet_raar 突崩（support 剧变→相位发散）即此类。**十一测已实测（[十一测汇报.md](十一测汇报.md)，C19）**：自适应 σ 4/4 消除 unet_raar 中途突崩（min ssim 全在 it0-100 收敛前），对症成功；但"防崩"≠"提质"——unet_raar 中心质量仍不如 RAAR（ssim 0.77–0.93 vs 0.94–0.96）。RAAR 自适应 σ 下仍 4/4 稳、HIO 10000 轮仍崩（support 改进救不了 HIO 震荡）。配合 similarity score 在多 PR 间引导（超单图范畴，暂不引入）。

## 4. 未训练神经网络的相位恢复（Deep Image Prior 路线）

### 4.1 动机与思路

传统 HIO 依赖手工设计的实空间约束算子（支撑域、直方图匹配等）。**深度图像先验**（Deep Image Prior, DIP）[10] 发现：未经训练的神经网络，其**结构本身**就是一个强先验——自然图像能被网络轻松拟合，而噪声/无结构信号很难。因此可以用一个**随机初始化、不喂任何训练集**的网络，通过拟合单张观测（如傅里叶振幅），让网络自然地"过滤"出符合先验的解。

把这一思想用到相位恢复：用一个未训练网络 G(z;θ)（z 为固定随机潜码，θ 为待优化权重）生成实空间图，约束其傅里叶振幅匹配实验值，反向传播优化 θ。这就是本项目乙方 UNet 方案的理论依据。

### 4.2 ADMM Net-ADM 方案（Ma et al. 2024）

本地文献中与本项目最直接对应的是 [19]：

**网络结构**：类 Deep Decoder（非 UNet），每层 = 1×1 卷积 + ReLU + 通道归一化 + 双线性上采样，末层 Sigmoid 输出 [0,1]。欠参数化（参数数 ≪ 图像维度），架构即正则。

**优化目标**：

$$\min_\theta \mathcal{L}(G(z;\theta)), \quad \mathcal{L}(v) = \tfrac{1}{2}\big\|\,|F_{\text{orig}}| - |\mathcal{F}v|\,\big\|^2 \quad (\text{Fourier 振幅 MSE})$$

为保证梯度 Lipschitz，用平滑版：

$$\mathcal{L}_\beta(v) = \tfrac{1}{2}\big\|\sqrt{|F_{\text{orig}}|^2+\varepsilon} - \sqrt{|\mathcal{F}v|^2+\varepsilon}\big\|^2$$

**未训练先验**在网络中以**约束**形式出现：x 必须落在网络的生成空间 Range(G) 内，通过 ADMM 的投影实现：

$$\mathcal{L}_\rho(x,v,\lambda) = \mathcal{L}_\beta(x) + \langle\lambda,\, x-G(v;\theta)\rangle + \tfrac{\rho}{2}\|x-G(v;\theta)\|^2$$

**迭代闭环**（Algorithm 1）：外环 ADMM（x 更新满足振幅保真，用 FFT/IFFT 显式求解；λ 对偶更新），内环对 θ 做梯度下降让 G(z;θ) 逼近 x。单图自监督，无训练集。

### 4.3 与监督学习方案的对比

本地文献另有两条**监督学习**相位恢复路线，需大规模训练集，与本项目"单图未训练"思路不同：

- **Pan et al. 2023** [20]：3D 卷积 U-Net + 残差 + SE 块，从 Patterson map 预测电子密度图，损失 = MSE + λ(1−Pearson CC)，输出 tanh 归一化到 [−1,1]。
- **PhAI (Larsen et al. 2024)** [21]：从 |F| 直接预测相位 φ，训练集 4900 万人工结构，相位循环（phase recycling）迭代收敛。

| 维度 | Pan 2023 [20] | PhAI 2024 [21] | **Net-ADM [19] / 本项目乙方** |
|---|---|---|---|
| 训练方式 | 监督 | 监督（4900 万结构） | **自监督（单图，无训练集）** |
| 输入 | Patterson map | \|F\| + 初始相位 | Fourier 振幅 |
| 网络 | 3D U-Net | 3D Conv + MLP | UNet / Deep Decoder |
| 先验来源 | 训练数据分布 | 训练数据分布 | **网络架构本身** |

> 本项目乙方采用 **UNet**（用户指定）替代 Deep Decoder，但"未训练、单图自监督、振幅 MSE 损失、sigmoid 输出约束 [0,1]"的核心设计与 Net-ADM 一致。

此外本地还有相关工作：李董鑫 [15] 用 encoder-decoder CNN 从衍射图回归物体轮廓（损失 MSE + SSIM）；郭梦霖 [16] 用深度卷积网络做衍射数据缺失恢复；方恒睿 [17] 研究 partial structure；周涛 [18] 用深度学习做衍射图样空间群分类；he 2015 [22]、he 2025 [23] 研究直进法（direct phasing）。

## 5. 相位恢复评估指标

| 指标 | 定义 / 公式 | 含义 / 阈值 | 出处 |
|---|---|---|---|
| **振幅域相关系数 CC** | $\dfrac{\sum F_{\text{obs}}F_{\text{cal}}\cos\Delta\varphi}{\sqrt{\sum F_{\text{obs}}^2 \cdot \sum F_{\text{cal}}^2}}$ | 主指标，>0.8 视为收敛 | [12,20] |
| **平均相位误差 Δφ** | $\big\langle\arccos(\cos(\varphi_{\text{true}}-\varphi_{\text{calc}}))\big\rangle$（arccos·cos 防 2π 卷绕） | <60° 良好 | [12,15,20] |
| **R 因子** | $R=\sum\|F_o-kF_c\|/\sum\|F_o\|$ | 模型残差，<20% 良好（晶体学专用） | [12] |
| **PSNR** | $10\log_{10}(\text{MAX}^2/\text{MSE})$ | 图像质量通用标准 | [15,19] |
| **SSIM** | $[l(x,y)\cdot c(x,y)\cdot s(x,y)]$（亮度×对比度×结构） | 结构相似性，符合人眼感知 | [15,19] |
| **Pearson CC** | $\dfrac{\sum(x_i-\bar x)(y_i-\bar y)}{\sqrt{\sum(x_i-\bar x)^2\sum(y_i-\bar y)^2}}$ | 线性相关性 | [20,21] |
| **支撑域匹配率 IoU** | 恢复 mask 与真实 mask 的交并比 | 结构重合度 | [12] |
| **SNR（噪声水平）** | $20\log_{10}\big[\text{Var}(|Fx|)/\sigma^2\big]$ | 噪声水平定义 | [19] |

### 5.1 指标使用前提：平凡歧义与配准（★ 四测修正）

上表指标直接使用前有一个**容易被忽略的前提**。相位恢复存在固有的**平凡歧义**（trivial ambiguities）：恢复出的物体可以整体**平移**、**共轭反转（twin image）**、乘一个**全局相位因子**，而傅里叶振幅谱完全不变 [28,29]。因此未配准时多个指标会被平移严重污染：

- **Δφ ≈ π/2 不一定是"相位随机"**：物体整体平移 x₀ 时，频域相位差 = 2πk·x₀（线性相位），`arccos(cos(·))` 把它折叠成三角波，**均值恒等于 π/2**；
- **amp_cc ≈ 0 同理**：cos(2πk·x₀) 在频域被振幅加权求和后正负抵消；
- **ssim / psnr / pearson_cc / support_iou** 全是空间逐像素比较，整体平移后必然暴跌。

换言之，Δφ 卡 π/2 + amp_cc≈0 **是"物体整体平移"的特征签名**，与"相位完全没恢复"在指标上无法区分。仅凭这些指标下"相位没恢复"的结论不可靠——这正是四测初判被推翻的原因（见 [实验猜想与结论.md](实验猜想与结论.md)）。

此外 **ssim / psnr 在暗背景图上易虚高**：塌缩到近零/平坦的图反而可能拿到高 ssim/psnr（C4），绝不能单独作判据。

### 5.2 评估前的歧义群配准

正确做法是在算指标前，先在歧义群上把恢复图对齐到真值：

1. **歧义群**：{ ±1 整体符号 } × { 正 / 180° 共轭反转 } × { 任意整数平移 }——这 2×2=4 类变体的振幅谱都与原 |F| 相同 [28,29]；
2. **求最佳平移**：对每个变体用 **FFT 域归一化互相关**（相位相关法 [30]）求使互相关峰值最大的整数平移；
3. **取最优**：4 个变体里互相关峰值最高者，作为对齐后的恢复图；
4. **投影再算**：对齐后 `ρ_eval = max(ρ, 0) × support_pred`，再统一计算全部 6 个指标。

配准消除了线性相位后，**amp_cc / Δφ 才真正反映相位的本质恢复程度**，空间类指标也回到公平的逐像素比较。

### 5.3 多次重建对齐平均（抑制随机条纹，★ 八测方向）

单次相位恢复重建不具代表性：不同随机初始相位收敛到不同局部解、产生不同的随机条纹 [33]。标准做法（[33] §3.5）：

1. 跑 **N 次独立重建**（不同随机初始相位，如 N=100）；
2. 按 error 函数选 **error 最小的 M 张**（如 M=10）；
3. 每张相对参考做**歧义群配准**（平移 + 共轭翻转 + 符号枚举 + FFT 互相关，同 §5.2）；
4. 配准后做**像素平均**；
5. 平均图再出图 / 评估。

**原理**：随机成分（周期条纹、噪声）每次不同，平均后趋零；真实物体稳定，平均后保留 → SNR 提升。本项目七测的 50 个实验就是现成多次重建，可离线对每类（HIO 30 张 / tanh+HIO 10 张 / sigmoid 10 张）配准平均，验证斜线能否被平均掉。

### 5.4 PRTF：客观收敛判据与分辨率（★ 八测方向）

**相位恢复传递函数**（Phase Retrieval Transfer Function, PRTF）[33]：

$$\text{PRTF}(u) = \frac{\langle |\Gamma(u)| \rangle}{|F(u)|}$$

其中 |F(u)| 为测量振幅，|Γ(u)| 为 **M 张配准重建图平均后**的 FFT 振幅，⟨·⟩ 表对多次重建平均。

- **PRTF ≈ 1**：多次重建的相位彼此一致、振幅匹配 → **真收敛**；
- **PRTF ≈ 0**：相位在多次重建间随机 → **未收敛**；
- PRTF 随频率衰减，**衰减到 1/e 或 0.5 的频率点定义为重建分辨率**。

**对本项目的用途**：① **客观收敛判据**——HIO 若未真收敛，PRTF 高频迅速跌 0；sigmoid 相变组 PRTF 保持高，从而无需人眼看图即可判收敛（替代 ssim，七测遗留）；② **分辨率定义**。PRTF 需多次重建（七测数据现成）。

## 6. 本项目对比实验方案

### 6.1 实验设计（控制变量法）

在同一张图 567.png 上对比甲方（HIO）与乙方（未训练 UNet）。**唯一变量是像空间处理**：

- **甲方像空间**：FFT 后用 |F_orig| **硬替换**新振幅，保留新相位；
- **乙方像空间**：UNet 输出做 FFT 取振幅，与 |F_orig| 算 **MSE 损失反传**更新网络（软学习）。

甲乙双方的**实空间约束完全一致**，从而隔离出"像空间处理方式"这一变量的影响。

### 6.2 双方统一的实空间约束

| 约束 | 采用 | 说明 |
|---|---|---|
| 正值 ρ≥0 | ✅ | 电子密度物理约束 |
| 实数约束 | ✅ | IFFT 取实部 |
| 直方图匹配 HM | ✅ | Zhang-Main 分箱线性映射，参考直方图**从原图统计**，只在 support 内做 |
| 支撑域 support | ✅ | **shrinkwrap 动态估计**（不偷看原图） |
| 溶剂平摊 | ❌ | support 约束已覆盖 |
| NCS 平均 | ❌ | 图像无对称性 |
| 密度限幅 | ❌ | 不采用 |

### 6.3 甲方配置（严格 HIO）

- β = 0.7，公式见 §2.2；
- 一轮迭代：ρ_k → FFT → 替换振幅 → IFFT → ρ′_k → [HIO 实空间约束 + HM] → ρ_{k+1}；
- shrinkwrap 每 N 轮更新 support。

### 6.4 乙方配置（未训练 UNet）

- UNet 随机初始化、未训练、单图自监督、sigmoid 输出 [0,1]；
- 一轮迭代：ρ_k → [实空间约束] → UNet → y → FFT(y) → MSE(|F_y|, |F_orig|) → 反传更新 UNet → ρ_{k+1}=y；
- 第一轮按 HIO 顺序"先实空间"起手。

### 6.5 评估指标（全套）

PSNR、SSIM、Pearson CC、振幅域 CC、平均相位误差 Δφ、支撑域匹配率 IoU、收敛曲线（各指标 vs 迭代轮数）。

- Python 解释器：`D:\anaconda3\envs\use\python.exe`
- 依赖：PyTorch（`torch.fft` 做可微 FFT/IFFT）、numpy、opencv-python、matplotlib。
