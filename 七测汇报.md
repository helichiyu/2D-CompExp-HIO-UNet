# 七测汇报

> 七测做**大规模重复取统计**：10 组 × 5 实验 = 50 个（HIO 5000 轮 ×3 + UNet 1500 轮 ×2），把六测 C12（三种方法概率恢复）从"方向性"升级为**统计性结论**。最强发现：**sigmoid 相变组质量最好（曲线丝滑、完全恢复）**；**HIO 能恢复物体但长迭代后几乎必然出周期斜线**，文献查到主因是 **FFT 卷绕效应**。理论见 [调研报告.md](调研报告.md)，猜想演变见 [实验猜想与结论.md](实验猜想与结论.md)，历史见 [初测](初测汇报.md)、[二测](二测汇报.md)、[四测](四测汇报.md)、[五测](五测汇报.md)、[六测汇报.md](六测汇报.md)。

## 1. 目标与动机

六测核心遗留（C12）：三种方法的恢复都是概率性事件，单次结论非稳健，需多次重复取统计。七测即 50 个实验取命中率/统计。附带验证六测张力（HIO 加轮次能否收敛）。

## 2. 实验

10 组 × 5 实验，HIO 5000 轮 ×3 + UNet（tanh+HIO / sigmoid）1500 轮 ×2。组间差异完全来自随机性（phase seed 不固定 + CUDA 非确定性 C6），即 10 次独立重复。代码改造：每个实验完整保存 + 断点续跑 + summary 汇总（见 [七测计划.md](七测计划.md)）。

## 4. 看图结论（用户确认，以图为准 R8）

1. **sigmoid 质量最好**（修正指标误导）：相变组 g6/g8 完全恢复、收敛曲线**丝滑无震荡**；tanh+HIO 的 ssim 0.93+ **虚高**（实际只"一点轮廓"）。按质量：sigmoid 相变 > tanh+HIO 轮廓 > HIO 带斜线。
2. **tanh+HIO 大多数轮效果不好**：7/10 有轮廓但未完全恢复，曲线始终震荡，PSNR 部分组跌落（与 γ/HIO 公式相关）。
3. **HIO 认知大推翻**：HIO **能恢复物体**（30 张图中间都有良好图像），但**长迭代后几乎必然出现周期斜线**，斜线周期/倾角每次不同。
4. **六测更正**：用户六测曾把 tanh/sigmoid 序号看反；根本结论 sigmoid 更好（与七测一致）。

## 5. 周期斜线文献调查（★ 七测新增核心）

### 5.1 主因：HIO 负反馈震荡 / stagnation（修正：非 padding/卷绕）
HIO 在 Fourier 空间用负反馈逼近 support 约束，β/γ 不当时会**震荡 / 停滞（stagnation）到周期态**，实空间表现为周期条纹/斜线（[Latychevskaia 2018](https://arxiv.org/abs/1809.04626) §3.4/§2.3.2；[Wikipedia HIO](https://en.wikipedia.org/wiki/Hybrid_input-output_algorithm)）。**七测强证**：sigmoid（无 HIO 反馈）收敛曲线丝滑无震荡、不出斜线；HIO / tanh+HIO（有 HIO 反馈）曲线始终震荡、出斜线。每次相位随机 → 条纹周期/倾角随机。

> **纠正前期误判**：曾据 wrapping effect 推断"padding 不足致卷绕严重"。但**物体是 567.png 中间一小块**，在 1216×608 画布中 padding **充足**，wrapping 影响应小。故主因归 HIO 震荡，**加大 padding 不作为八测方向**。

### 5.2 次因（存疑）：FFT 卷绕效应
[Latychevskaia 2018 §3.6.2](https://arxiv.org/abs/1809.04626)：物体谱无限，FFT 谱有限致超出 Nyquist 的高频卷绕到谱边缘，实空间成条纹。原文明确仅"物体边缘信号强 + padding 不足"时显著；本项目 padding 充足，wrapping 非主因。

### 5.3 三方法差异的自洽解释
- **sigmoid 最好**：无 HIO 反馈震荡 + UNet 平滑先验天然抑制高频条纹 → 相变后干净、曲线丝滑。
- **HIO 最差**：HIO 震荡 + 长迭代条纹累积 → 物体在中间但被斜线淹没。
- **tanh+HIO 居中**：有 HIO 震荡（曲线震荡、PSNR 漂移）但 UNet 部分抑制。

### 5.4 文献解法（八测方向）
1. **ER + HIO 交替**（Latychevskaia §3.4，避免震荡/停滞——主攻方向）；
2. **多次重建对齐平均**（§3.5，抑制随机条纹，七测 50 实验现成可离线试）；
3. **PRTF 客观评估**（§3.5，收敛判据 + 分辨率，替代人眼看图）；
4. ~~加大 zero-padding~~（撤回：padding 已充足，wrapping 非主因）；
5. tanh+HIO 震荡 / PSNR 跌落（C8 延续，查 γ/β）。

## 6. 关键认知（七测核心）

### 6.1 C12 升级为统计性结论（带命中率）
三种方法概率恢复，现有了硬统计：
- HIO 0/30、tanh+HIO 7/10（但虚高）、sigmoid 2/10（质量最好）。
- **多次重复取统计是必须的**（R10）——单次会误判（六测 tanh+HIO 单次没恢复，七测 70% 有轮廓；但看图又发现轮廓只是虚高）。

### 6.2 C2/C10 大推翻：HIO 能恢复，但长迭代必出周期斜线
不是"概率性收敛"（C10 旧表述），而是"**HIO 能恢复物体 + 长迭代几乎必然出周期斜线（卷绕）**"。斜线与迭代长度**正相关**（长迭代条纹累积），与 C10"与迭代长度无单调关系"相反。

### 6.3 C13（新）：sigmoid 质量最佳 + UNet 平滑先验抑制卷绕
sigmoid 相变组完全恢复、曲线丝滑，是三方法中**单次质量最高**的。机理：无 HIO 震荡 + UNet 平滑先验抑制高频卷绕条纹。但相变是低概率事件（20%）。

### 6.4 指标虚高再升级（R5/R8 强化）
不仅 ssim，连 **iou/amp_cc 高的 tanh+HIO 也只是"一点轮廓"**。指标横向比较不可靠，**必须看图**。sigmoid 的"曲线丝滑"才是真收敛的可靠信号。

### 6.5 周期斜线机理明确（C14 新）
FFT 卷绕效应（+ padding 不足 + HIO 震荡）。八测可针对解法验证。
